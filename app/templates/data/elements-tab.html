Übersicht zu Bedienelementen

Bedienelemente sind die Stage, die Fader und die Buttons. Auf der Stage sind die einzelnen Geräte zu bedienen. Fader und Buttons bedienen Szenen (=Cues).

Die Parametrierung der Fader und Buttons ist in einzelnen Dateien gespeichert. Durch die Auswahl der Dateien wird festgelegt, welche Cues zur Bedienung zur Verfügung gestellt werden.

Start-Optionen

Es gibt drei Möglichkeiten, wie der Start von ClubDMX erfolgen kann. Die Basisvariante ist, dass alle Fader und Buttons auf Level 0 gesetzt werden. Es sind also keine Cues aktiv.

In Variante 2 werden die Levels und der Status der Buttons vom letzten Verwenden von ClubDMX wiederhergestellt. Dazu muss das Speichern der Levels aktiviert werden: Levels speichern auf ein

In Variante 3 wird ein Startcue festgelegt, der zum Programmstart aktiviert wird. Dieser Startcue muss entweder als Button oder als Fader angelegt sein, damit er auch beim Start gefunden wird. Starte mit Startcue auf ein, einen Startcue auswählen.
